# Notebook Option

This project is script-first so it can run end-to-end with:

```bash
python run_pipeline.py
```

If you want notebooks later, create these from the scripts:

1. `01_data_generation_and_eda.ipynb`
2. `02_demand_forecasting.ipynb`
3. `03_inventory_optimization.ipynb`
4. `04_scenario_analysis.ipynb`
5. `05_powerbi_export.ipynb`
